# Linear grammars and the linear languages

## Statement

A **linear grammar** is a context-free grammar in which every rule's right-hand side
contains **at most one nonterminal** (so each sentential form has a single
nonterminal). The languages they generate, the **linear languages**, sit strictly
between the regular and the context-free languages:

> regular ⊊ **linear** ⊆ context-free.

The witness for the first strict inclusion is `{aⁿbⁿ}`, which is linear (one
nonterminal grows the word symmetrically from the middle) but not regular.

## Proof idea

A linear grammar keeps exactly one nonterminal "cursor" in the sentential form, so a
derivation is a single spine — unlike a general context-free grammar, which can
branch into several nonterminals. Regular grammars are the special case where the
nonterminal only ever sits at one end, which gives `regular ⊆ linear`; `{aⁿbⁿ}`,
generated by `S → aSb | ε`, is linear but (by the pumping lemma) not regular, making
the inclusion strict. Dropping the at-most-one-nonterminal restriction recovers the
full context-free grammars, giving `linear ⊆ context-free`.
